# horikawa-SMPlab.github.io
軟X線分光を用いた液体、溶液中の有機分子の電子状態研究を行っています。 測定手法の開発も行いながら、ソフトマター分野への軟X線分光の応用も進めています。（I study the electronic states of organic molecules in liquids and solutions using soft X-ray spectroscopy. In parallel with developing measurement techniques, I am also advancing the application of soft X-ray spectroscopy to the field of soft matter.）
